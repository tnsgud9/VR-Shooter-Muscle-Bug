=============
### 팀명 : 불타는 핑크 (목요일 정규반)
### 게임명 : Muscle Bug
### 지원 플랫폼 : PC, Steam VR

### 개발 환경 : Windows 10, Unity 2019 2.0

### < 게임 소개 >
#### 시놉시스
방역 회사 직원인 주인공은 한 제약공장의 벌레 청소의뢰를 받고 공장으로 간다. 하지만 약의 부작용으로 인한 벌레들이 주인공을 공격하러 오는데..

#### 플레이 방법
트리거 버튼을 통해 다가오는 적들을 쏘면 된다.
